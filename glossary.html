Glossary

HTML
HyperText Markup Language – the standard language used to create web pages.
CSS
Cascading Style Sheets – used to style and layout HTML elements on a web page.
ART DIRECTION
Constructing HTML so that different images are used at different screen sizes is known as art direction
SVG Image
SVG, or Scalable Vector Graphics is an XML format for describing vector images.
CAPTION
Captions – provide a transcription of audio for those who cannot hear
